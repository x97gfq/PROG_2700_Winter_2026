What is jQuery UI?

jQuery UI is a curated set of user interface interactions, effects, widgets, and themes built on top of jQuery.

Why use jQuery UI?

It provides ready-to-use widgets that are accessible, themeable, and easy to implement in your projects.

Common Use Cases

Date pickers, drag and drop, sortable lists, dialogs, and much more!
