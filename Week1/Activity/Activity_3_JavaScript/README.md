# Activity 3 - JavaScript Basics

## Overview
This activity introduces JavaScript by creating an interactive random word generator.

## Files
- **index.html** - HTML structure with button and readonly input (linked to empty `script.js`)
- **solution/index.html** - Same HTML structure
- **solution/script.js** - Complete JavaScript implementation

## Concepts Covered
- DOM manipulation (`querySelector`, `getElementById`)
- Event listeners (`addEventListener`)
- Functions
- Arrays and random number generation
- Updating element values

## Instructions
1. Open `index.html` - the button won't work (no JavaScript yet)
2. Open `solution/index.html` - click the button to generate random Gen Z slang words!
3. Review `solution/script.js` to see how it works
