# Activity 2 - CSS Styling

## Overview
This activity demonstrates the power of CSS by showing a before/after comparison of the same HTML page.

## Files
- **index.html** - Barebones HTML with no styling (linked to empty `styles.css`)
- **solution/index.html** - Same HTML structure with CSS classes applied
- **solution/styles.css** - Complete stylesheet with 5 classes demonstrating basic CSS concepts

## Concepts Covered
- Class selectors
- Basic styling (colors, padding, margins, borders)
- Card-based layout
- Form element styling

## Instructions
1. Open `index.html` to see the unstyled version
2. Open `solution/index.html` to see the styled version
3. Compare the visual difference that CSS makes!
