
# Activity 2 — Selector & Logic Micro‑Drill (15–20 min)
**Brand:** PROG 2700 Client-Side Programming

## Objectives
- Target elements with attribute/class selectors
- Practice variables and `if/else`

## Materials
- `styles.css`
- `script.js` (snippet template)

## Setup
1. Open `index.html` in your browser.
2. Link `styles.css` and `script.js` (already linked).

## Guided Steps
1. **CSS:** Write an attribute selector rule, e.g. `input[type=email] { /* styles */ }`.
2. **JS:** Declare 3 variables and build a simple conditional to toggle `.error`.
3. **Test:** Try valid/invalid input and observe styling changes.

## Teacher Prompts (ask live)
- When would you prefer class vs attribute selectors?
- Where should validation run: on blur or on submit?

## Wrap‑Up
- Save examples for Friday’s lab.
- Note one selector gotcha.
